# Scala 3 and Functional Programming Essentials

## Takeaways & Notes
### Chapter 1
- If you need interoperability or cross compilation between Scala 2 and Scala 3, it's recommended to define a `main` method inside an `object` value.
- Avoid using the `App` trait, since it's deprecated since __. More information [here](https://docs.scala-lang.org/scala3/book/methods-main-methods.html).
